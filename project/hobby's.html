Mijn hobby's/Interesses

Gaming

Ik ben van kleins af aan al een best grote gamer. Ik ben echt geinterreseerd toen ik mijn eerste computer kreeg op mn 11ste verjaardag
Dit heeft mij vooral geholpen met Engels spreken en uiteindelijk ook meer een interesse te bouwen in Tech.
De main games die ik vooral speelde waren Shooters/Battle royale games zoals:
  • CSGO
  • H1Z1
  • COD
  • Overwatch
  • ...
Ik kan hier een heel lange lijst zetten maar dit is niet zo belangerijk.

Deze dagen speel ik vooral Final Fantasy 14 Online. Een MMORPG gebasseerd op de Final Fantasy series.
Ik heb deze game ontdekt ongeveer een 2 jaar geleden en heb er ondertussen een heel aantal uren ingestoken.
Dit is op de moment zeker mijn favoriete game en ik hoop dat dik ook zo nog wel een tijdje blijft.

Tech

Ik ben de laatste 3 jaar bezig met wat oude computers die ik van familie had gekregen om mee te experimenteren.
Hier test ik eigenlijk gewoon een heel aantal dingen waarvoor ik mijn main desktop niet voor wil gebruiken
Op deze moment heb ik bijvoorbeld een minecraft server lopen op een van deze oude desktops waar ik wat nieuwe onderdelen had ingestoken.

Ik had dit geprobeerd met de oude onderdelen maar de mods die op de server stonden ingesteld vroegen voor meer geheugen dus de server was niet echt speelbaar.
De fun in dit is wanneer alles werkt, dat gevoel blijft leuk.
